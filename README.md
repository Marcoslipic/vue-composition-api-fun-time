# clickerapp
This is a basic vue 3 app. It uses the composition API to increment, decrement, and double an integer.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
